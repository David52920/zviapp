Gas Analysis Calculator

This calculator is only available in the valvesizing window.

Adding/Removing Component from Gas List

User is able to add custom gas components; however they will not be able to be used as the final gas mixture; as the .FLD files may not be present.

Adding Component

To add a component simply click the '+' button

Add the component name via dialog.

Choose the correct molecular weight for the component and click 'Ok'

The newly added component should be available in the gas list as shown above.

Removing Component

To add a component simply select the component and click the '-' button

Running Calculator

User can select either one or multiple components (hold ctrl key) from the availble gas list.

'<<<' removes components from chosen table and '>>>' adds components to the chosen table

Start by entering each components Mol% in the corresponding column of the 'Chosen Gases' table.

User must select more than 1 component from the 'Chosen Gases' table to be able to run the calculator

Once finished; user can add the set to use in valvesizing by using the 'Add Set' button as shown.

Set can be viewed and selected using the dropdown
