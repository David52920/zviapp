How to start stage matching

To use stage matching; the number of stages must be more than 1

Click the 'Stage Matching' icon and a popup will show to input deltas per stage

Once finished; click 'Submit' and stage matching will be performed on all selected throws.

Progress will be shown in bottom right corner of the valvesizing window.

If out-of-range; you will be prompted to change or use previous delta values.
