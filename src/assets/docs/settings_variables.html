Changing Application Variables

All items in the value column can be changed simply by double clicking and changing the value.

'Local' and 'Server' button changes the variable file that is being used for the entire application.

Local is users version of the startup files and Server is standard across all users.

User can filter out variables via the filter text area.

There 'Revert' button simply reverts every application variable from the specified startup file.

There 'Undo' & 'Redo' works only on recent changes and not the entire list of variables.
